# 📖 Drumify Documentation

Chào mừng bạn đến với hệ thống tài liệu của Drumify. Hệ thống này được thiết kế để cung cấp thông tin nhanh chóng và chính xác cho các AI Agent và Developer.

## 📂 Thư mục tài liệu

- **`backend-plans/`**: Chứa các bản kế hoạch chi tiết cho các feature backend.
- **`frontend-plans/`**: Chứa các bản kế hoạch chi tiết cho các feature frontend.
- **`design-briefs/`**: Các yêu cầu về thiết kế, trải nghiệm người dùng.
- **`ideas/`**: Nơi lưu trữ các ý tưởng mới, bản nháp và `SESSION_CONTEXT.md`.
- **`ui-mockups/`**: Các mô tả hoặc hình ảnh về giao diện người dùng.

## 📄 Các file quan trọng

- **`ARCHITECTURE.md`**: Tổng quan về kiến trúc hệ thống, tech stack và luồng bảo mật.
- **`STYLEGUIDE.md`**: Quy chuẩn về thiết kế (MUI, Glassmorphism) và đặt tên code.
- **`FEATURES_DONE.md`**: Danh sách các tính năng đã hoàn thành để AI biết những gì đã có sẵn.

---
*Tài liệu này là "Single Source of Truth" cho dự án Drumify.*
