# 💡 Project Ideas & Future Features

Nơi lưu trữ các ý tưởng và dự định phát triển cho Drumify.

## 🚀 Tính năng đề xuất
- **Hệ thống Loyalty**: Tích điểm cho khách hàng thân thiết.
- **Tích hợp thêm đơn vị vận chuyển**: Giao Hàng Tiết Kiệm, Viettel Post.
- **Mobile Web Optimization**: Đảm bảo trải nghiệm mượt mà trên trình duyệt di động.
- **AI Recommendation**: Gợi ý sản phẩm dựa trên sở thích người dùng.

## 🛠️ Cải tiến kỹ thuật
- **Unit Testing**: Tăng độ bao phủ test cho backend.
- **Caching**: Sử dụng Redis để tối ưu tốc độ truy xuất sản phẩm.
- **Microservices**: Cân nhắc tách module Warehouse và Order khi hệ thống lớn mạnh.
