# 🖼️ UI Mockups & Layouts Overview

Tài liệu này mô tả cấu trúc các màn hình chính của Drumify.

## 🏢 Admin Dashboard
- **Sidebar**: Bên trái, chứa menu điều hướng (Kho, Nhân sự, Đơn hàng, Báo cáo).
- **TopNav**: Chứa thông tin User, Nút Logout, Breadcrumbs.
- **Main Area**: Workspace hiển thị nội dung theo Tab.

## 📦 Warehouse Workspace
- **Tabs**: Sản phẩm, Danh mục, Nhãn hiệu, Nhà cung cấp.
- **Table View**: Danh sách các thực thể với nút sửa/xóa/thay đổi trạng thái.
- **Detail Modal**: Hiển thị thông tin chi tiết và form chỉnh sửa.

## 🛒 Checkout Page
- **Left Side**: Thông tin vận chuyển, địa chỉ, phương thức thanh toán.
- **Right Side**: Tóm tắt đơn hàng (Giỏ hàng), tổng tiền.

## 🏠 Home Page
- **Hero Section**: Banner nổi bật.
- **Product Grid**: Hiển thị danh sách sản phẩm với Filter Sidebar bên trái.
